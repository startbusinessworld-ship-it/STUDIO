# SBW Studio

Planning éditorial · Scripts · Téléprompter vocal

## Déploiement

Ce projet est déployé automatiquement sur Vercel via GitHub.

## Mot de passe

Mot de passe par défaut : `sbw2025`

Pour le changer : ouvrir `index.html` et modifier la ligne :
```js
const PASSWORD = 'sbw2025';
```
